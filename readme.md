## 🎯 Number Guessing Game

A simple and interactive number guessing game built using HTML, CSS, and JavaScript.  
Test your guessing skills and try to achieve the best score!

## 🚀 Features
- Random number generation between 1 and 100
- User input validation
- "Too High" and "Too Low" hints
- Attempts counter
- Best score tracking (using localStorage)
- Restart game functionality
- Clean and responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
    index.html
    style.css
    script.js


## 🎮 How to Play
1. Enter a number between 1 and 100
2. Click "Submit Guess"
3. Get hints:
   - Too High
   - Too Low
4. Keep guessing until you find the correct number
5. Try to beat your best score!
